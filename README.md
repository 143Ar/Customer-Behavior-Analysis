# Customer-Behavior-Analysis
Project Overview

This project focuses on analyzing customer behavior using a structured data analysis pipeline. The goal is to extract meaningful insights from customer data to understand purchasing patterns, preferences, and trends.The project integrates data cleaning, SQL-based analysis, and interactive visualization to deliver business insights.

🚀Tech Stack
Python (Jupyter Notebook) – Data cleaning & preprocessing
MySQL Workbench – Data storage & SQL analysis
Power BI – Data visualization & dashboard creation
🔄 Workflow
1️⃣ Data Cleaning (Jupyter Notebook)
Handled missing values
Removed duplicates
Standardized data formats
Prepared dataset for analysis
2️⃣ Data Storage & Querying (MySQL)
Imported cleaned data into MySQL
Wrote SQL queries to analyze:
Customer purchase patterns
Most popular product categories
Customer segmentation
Order frequency and trends
3️⃣ Data Visualization (Power BI)
Built interactive dashboards
Visualized key metrics like:
Sales trends
Category-wise performance
Customer behavior insights
Created filters and slicers for dynamic analysis
📊 Key Insights
Identified top-performing product categories
Analyzed customer purchasing frequency
Discovered trends in customer preferences
Derived actionable insights for business decision-making
